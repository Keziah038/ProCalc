# ProCalc
ProCalc – Compact Animated Calculator

ProCalc is a stylish, compact, and feature-rich calculator web app built using HTML, CSS, and JavaScript. It includes animated backgrounds, multiple themes, scientific calculations, memory functions, voice feedback, history tracking, and a sleek modern UI.

## Features
#### Basic Calculator Functions
* Addition, subtraction, multiplication, and division
* Decimal support
* Percentage calculations
* Positive / negative toggle
* Delete last digit
* Clear all function
#### Scientific Functions
* sin, cos, tan
* Square root
* Square and cube
* Power function
* Logarithm and natural log
* Pi and Euler's number
* Absolute value
* Factorial
* DEG / RAD angle mode toggle
#### Advanced Features
* Animated gradient background
* Floating particle effects
* Multiple themes:Light,Dark,Aurora,Sunset,Midnight,Cherry
* Live clock and date display
* Voice feedback using Speech Synthesis API
* Copy result to clipboard
* Memory functions:M+,M-,MR,MC
* Calculation history with timestamps
* Export history as text file
* Local storage support for:Theme preference,Calculation history
* Keyboard support
* Fully responsive design
  
#### Technologies Used
* HTML5
* CSS3
* JavaScript (Vanilla JS)
*Font Awesome Icons
* Local Storage API
* Speech Synthesis API
* Folder Structure
